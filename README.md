# Sistema PDV — Gestão de Bar

Sistema de ponto de venda (PDV) desenvolvido para gestão completa de um bar — do balcão ao fechamento de caixa.

> 🔒 **Projeto privado a pedido do cliente.** Este repositório apresenta apenas capturas de tela, descrição funcional e escopo técnico do projeto, sem exposição do código-fonte.

## Sobre o projeto

Modernização de um sistema de controle de vendas que originalmente rodava como uma ferramenta de terminal em Python, sem interface gráfica. O projeto evoluiu para uma aplicação com interface visual completa, inspirada no estilo clássico do Windows — familiar e intuitiva para o uso diário no balcão.

## Funcionalidades

- 🧾 Registro de vendas e comandas
- 💰 Controle de caixa (abertura, sangria, fechamento)
- 📦 Gestão de produtos e estoque
- 📊 Relatórios de vendas
- 🖥️ Interface gráfica com estética Windows clássica, pensada para uso rápido sem curva de aprendizado

## Meu papel no projeto

Responsável pela reformulação completa do sistema: migração de uma ferramenta de terminal para uma aplicação com interface gráfica, definição da identidade visual (réplica fiel da estética Windows), e implementação das funcionalidades de PDV, caixa e estoque.
